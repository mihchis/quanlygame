🔑 Cấu hình RAWG API

Ứng dụng cần API Key từ RAWG để lấy hình ảnh, mô tả và tìm kiếm game. API Key này là hoàn toàn miễn phí.

💻 Cấu hình Hệ thống (DxDiag)

Tải lên tệp tin dxdiag.txt của bạn để ứng dụng tự động đối chiếu cấu hình phần cứng khi xem thông tin game.

Chưa tải lên cấu hình
Hệ điều hành: -- CPU: -- RAM: -- GPU (Card): --

💾 Quản lý Dữ liệu

Lưu trữ hoặc khôi phục dữ liệu danh sách game của bạn (dưới dạng tệp tin JSON).

📘 Hướng dẫn lấy RAWG API Key miễn phí

  1. Truy cập trang web rawg.io và tạo một tài khoản (hoặc đăng nhập nếu đã có).
  2. Sau khi đăng nhập thành công, hãy truy cập trang rawg.io/apikeys.
  3. Nhấp vào nút "Create key" (hoặc Yêu cầu Key). Điền tên dự án bất kỳ (ví dụ: GameVault App) và mục đích sử dụng phi thương mại.
  4. Sao chép mã API Key nhận được và dán vào ô cấu hình ở trên để kích hoạt đầy đủ chức năng tìm kiếm game.
